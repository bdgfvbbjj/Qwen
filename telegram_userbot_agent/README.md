# Telegram Userbot AI Agent с Gemini

Полноценный AI-агент для управления вашим аккаунтом Telegram. Использует Gemini API для интеллектуального анализа запросов и может выполнять любые задачи в рабочей директории.

## 🚀 Возможности

### AI-функции:
- ✅ **Отвечает на ЛЮБЫЕ вопросы** используя Gemini AI
- ✅ **Анализирует фото, видео, голосовые сообщения** через мультимодальный Gemini
- ✅ **Контекстная память** - помнит историю переписки с каждым пользователем
- ✅ **Действует как человек** от вашего имени

### Файловые операции:
- ✅ Создание файлов по команде
- ✅ Чтение содержимого файлов
- ✅ Удаление файлов
- ✅ Поиск файлов по маске

### Выполнение кода:
- ✅ Shell команды (ls, grep, cat и т.д.)
- ✅ Python код
- ✅ Безопасное выполнение с ограничениями

### Telegram функции:
- ✅ Отправка файлов по запросу
- ✅ Отправка сообщений
- ✅ Обработка входящих медиа

## 📋 Требования

1. **Telegram credentials** (получить на [my.telegram.org](https://my.telegram.org)):

## 🔧 Настройка

Установите переменные окружения:

```bash
export TG_API_ID=ваш_api_id
export TG_API_HASH=ваш_api_hash
export TG_PHONE=+79991234567
export GEMINI_API_KEY=ваш_api_key

# Опционально - порядок переключения моделей при лимитах (ОТ ЛУЧШИХ К ХУДШИМ):
export GEMINI_MODELS=gemini-1.5-pro,gemini-2.0-flash-exp,gemini-1.5-flash

# Другие опции:
export ALLOWED_USERS=username1,username2  # разрешенные пользователи
```

### 🔄 Автоматическое переключение МОДЕЛЕЙ при лимитах

**ВАЖНО:** При исчерпании лимита одной модели агент автоматически переключается на другую!

По умолчанию используется порядок (от лучших к худшим):
1. `gemini-1.5-pro` - самая мощная и умная модель (используется первой)
2. `gemini-2.0-flash-exp` - экспериментальная быстрая модель (переключение при лимите)
3. `gemini-1.5-flash` - быстрая модель с высоким лимитом (последний вариант)

Вы можете изменить порядок или добавить другие модели:
```bash
export GEMINI_MODELS=gemini-1.5-pro,gemini-2.0-flash-exp,gemini-1.5-flash
```

Как это работает:
1. Агент начинает с ПЕРВОЙ (лучшей) модели в списке
2. При ошибке лимита (429, quota exceeded) автоматически переключается на СЛЕДУЮЩУЮ (более простую)
3. Когда все модели исчерпаны - ожидает восстановления лимитов
4. Используется ОДИН API ключ для всех моделей

### Пошаговая инструкция:

#### 1. Получение Telegram credentials:
1. Зайдите на https://my.telegram.org
2. Авторизуйтесь по номеру телефона
3. Перейдите в "API development tools"
4. Создайте новое приложение
5. Скопируйте **API ID** и **API Hash**

#### 2. Получение Gemini API ключа:
1. Зайдите на https://aistudio.google.com/app/apikey
2. Нажмите "Create API Key"
3. Скопируйте ключ

#### 3. Запуск:
```bash
python userbot_agent.py
```

При первом запуске бот запросит код подтверждения из Telegram.

## 🆘 Решение проблем

### Ошибка "Отсутствуют переменные окружения"
Проверьте что все переменные установлены:
```bash
echo $TG_API_ID
echo $GEMINI_API_KEY
echo $GEMINI_MODELS
```

### Ошибка авторизации Telegram
Убедитесь что номер телефона указан в международном формате: `+79991234567`

### Ошибка Gemini API - лимит модели
Агент автоматически переключится на следующую модель из списка GEMINI_MODELS.
Проверьте логи - должно быть сообщение о переключении.

### Все модели исчерпали лимиты
Подождите восстановления лимитов (обычно 1 час для gemini-2.0-flash, 24 часа для других).
Можно добавить больше моделей в GEMINI_MODELS или использовать другой API ключ.

### Бот не отвечает
Проверьте логи на наличие ошибок. Убедитесь что ваш username добавлен в `ALLOWED_USERS`.
