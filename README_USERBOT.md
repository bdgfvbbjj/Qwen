# Telegram Userbot Agent

Полнофункциональный юзербот для управления аккаунтом Telegram с возможностью выполнения задач в файловой системе.

## 🚀 Возможности

### Файловые операции
- Создание файлов в рабочей директории
- Чтение и просмотр содержимого файлов
- Удаление файлов
- Просмотр списка файлов в директориях
- Отправка файлов в чат

### Выполнение команд
- Выполнение shell команд через `/exec`
- Выполнение Python кода через `/python`

### Управление аккаунтом
- Получение информации об аккаунте
- Просмотр списка чатов
- Отправка сообщений в чаты
- Пересылка сообщений

## 📋 Требования

- Python 3.8+
- Библиотеки: `pyrogram`, `tgcrypto`
- API ID и API Hash от Telegram (получить на https://my.telegram.org)

## 🔧 Установка

1. Установите зависимости:
```bash
pip install pyrogram tgcrypto
```

2. Получите API credentials:
   - Зайдите на https://my.telegram.org
   - Авторизуйтесь по номеру телефона
   - Перейдите в "API development tools"
   - Создайте новое приложение и получите API ID и API Hash

3. Настройте переменные окружения:
```bash
export API_ID=ваш_api_id
export API_HASH=ваш_api_hash
```

Или отредактируйте файл `userbot_agent.py`, заменив значения по умолчанию.

## 🏃 Запуск

```bash
python userbot_agent.py
```

При первом запуске:
1. Введите номер телефона в международном формате (например, +79991234567)
2. Введите код подтверждения из Telegram
3. Если включена двухфакторная аутентификация, введите пароль

После успешной авторизации сессия сохранится в файле `userbot_agent.session`.

## 📖 Команды

| Команда | Описание |
|---------|----------|
| `/start` | Информация о боте |
| `/help` | Подробная справка |
| `/me` | Информация об аккаунте |
| `/status` | Статус бота |
| `/chats [limit]` | Список чатов |
| `/create_file <name> [content]` | Создать файл |
| `/read_file <path>` | Прочитать файл |
| `/delete_file <path>` | Удалить файл |
| `/list_files [path]` | Список файлов |
| `/send_file <path>` | Отправить файл в чат |
| `/exec <command>` | Выполнить shell команду |
| `/python <code>` | Выполнить Python код |
| `/send_msg <chat_id> <text>` | Отправить сообщение |

## 💡 Примеры использования

```
/create_file test.txt "Привет, мир!"
/exec ls -la
/python print(2 + 2)
/send_file test.txt
/send_msg @username Привет!
/chats 10
```

## ⚠️ Безопасность

- **Важно:** Юзербот работает от имени вашего аккаунта и имеет полный доступ ко всем функциям
- Не передавайте файл сессии (`*.session`) третьим лицам
- Используйте только с доверенными источниками команд
- Бот ограничен работой в своей рабочей директории для безопасности файлов
- Опасные shell команды блокируются
